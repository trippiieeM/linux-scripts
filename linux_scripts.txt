Files & directories

pwd — print current directory.
Example: pwd

ls — list files.
Example: ls -la

cd — change directory.
Example: cd ~/projects

mkdir — make directory.
Example: mkdir -p a/b/c

rmdir — remove empty directory.
Example: rmdir olddir

touch — create empty file / update timestamp.
Example: touch file.txt

cp — copy file/dir.
Example: cp file.txt file.bak / cp -r dir1 dir2

mv — move or rename.
Example: mv oldname newname

rm — remove file/dir.
Example: rm file.txt / rm -r folder/ (use with care)

Viewing file contents

cat — print whole file.
Example: cat README.md

less — scrollable viewer (q to quit).
Example: less /var/log/syslog

more — pager (older).

head — first N lines.
Example: head -n 20 file.log

tail — last N lines (use -f to follow).
Example: tail -f /var/log/syslog

nl — number lines.
Example: nl file.txt

Text processing & searching

grep — search for patterns.
Example: grep -n "error" *.log

egrep / grep -E — extended regex.

sed — stream editor (substitute, extract).
Example: sed -n '2,4p' file or sed 's/old/new/g' file

awk — field processing and reporting.
Example: awk '{print $1,$3}' file

cut — cut columns.
Example: cut -d':' -f1 /etc/passwd

sort — sort lines.
Example: sort names.txt

uniq — unique adjacent lines (often with sort).
Example: sort data.txt | uniq -c

tr — translate/replace characters.
Example: tr '[:lower:]' '[:upper:]' < file

wc — count lines/words/bytes.
Example: wc -l file.txt

File search

find — search filesystem by name, type, size, etc.
Example: find /home -name "*.py" -type f

locate — fast name search (uses DB; updatedb to refresh).
Example: locate .bashrc

Permissions & ownership

chmod — change permissions.
Example: chmod 644 file.txt or chmod u+x script.sh

chown — change owner/group.
Example: sudo chown user:group file

chgrp — change group.
Example: chgrp staff file

Users & groups

whoami — current username.

id — user & group IDs.

users — logged in users.

who / w — who is logged in and what they are doing.

useradd / adduser — create user (varies by distro).

passwd — change password.

sudo — run command as root.
Example: sudo apt update (use carefully)

Process & job control

ps — list processes.
Example: ps aux | grep chrome

top — interactive process viewer.

htop — improved top (if installed).

kill — send signal to PID (default SIGTERM).
Example: kill 1234 or kill -9 1234

pkill / killall — kill by name.
Example: pkill -f python

bg / fg / jobs — job control for background/foreground.

nice / renice — set/change process priority.

Networking basics

ip addr / ip a — show network interfaces & IPs.

ifconfig — older tool (may be absent on modern distros).

ping — test connectivity.
Example: ping -c 4 8.8.8.8

ss / netstat — view sockets & ports.
Example: ss -tuln

curl — HTTP client / download / test APIs.
Example: curl -I https://example.com

wget — download files.
Example: wget http://example.com/file.zip

ssh — remote shell.
Example: ssh user@host

scp / sftp — copy files securely over SSH.
Example: scp file.txt user@host:/path/

traceroute / tracepath — route path to host.

nslookup / dig — DNS queries.
Example: dig example.com

Disk & storage

df — filesystem disk usage (space).
Example: df -h

du — directory size.
Example: du -sh ~/projects

mount / umount — mount/umount filesystems.

lsblk — list block devices.

blkid — show filesystem UUIDs & types.

fdisk / parted — partitioning tools (advanced).

mkfs — create filesystem (advanced; destructive).

mount example: sudo mount /dev/sdb1 /mnt/usb

Compression & archives

tar — create/extract archives.
Examples:

Create: tar -cvf archive.tar dir/

Create compressed: tar -czvf archive.tar.gz dir/

Extract: tar -xzvf archive.tar.gz

gzip / gunzip — compress/decompress .gz files.

zip / unzip — .zip archives.

Package managers (distro-specific)

Debian/Ubuntu: apt update / apt install pkg / apt remove pkg

RHEL/CentOS: yum install pkg or dnf install pkg

Arch: pacman -S pkg

Fedora: dnf
Example: sudo apt update && sudo apt upgrade -y

System info & logs

uname -a — kernel & OS info.

lsb_release -a — distro info (if available).

uptime — system uptime/load.

free -h — memory usage.

dmesg — kernel ring buffer (boot messages).

journalctl — systemd logs (e.g., journalctl -u ssh or journalctl -b).

tail -f /var/log/syslog or /var/log/messages — live logs.

Shell & environment

echo — print text or variables.
Example: echo $HOME

env — print environment variables.

export — set environment variable.
Example: export PATH=$PATH:~/bin

alias — create command shortcuts.
Example: alias ll='ls -la'

history — show command history.

source / . — run commands from file in current shell.
Example: source ~/.bashrc

Jobs, scheduling & automation

crontab -e — edit cron jobs.

at — schedule one-time task (if installed).

systemctl — manage services on systemd systems.
Examples: sudo systemctl start nginx, sudo systemctl status ssh

Development & version control (basics)

git clone / git status / git add / git commit / git push
Example: git clone https://github.com/user/repo.git

Useful utilities

which — find executable in PATH.
Example: which python3

whereis — locate binary, source, manual.

file — detect file type.
Example: file archive.tar.gz

watch — run a command repeatedly.
Example: watch -n 2 df -h

xargs — build/execute commands from stdin.
Example: find . -name '*.log' | xargs grep -i error

tee — write output to file and stdout.
Example: ls -la | tee listing.txt

Recovery / low-level (use with extreme caution)

dd — copy/convert raw data (can wipe disks).
Example: dd if=/dev/sda of=image.img bs=4M (dangerous if misused)

fsck — filesystem check/repair (run when unmounted).

Quick examples of piping & redirection

Redirect stdout to file: command > out.txt

Append stdout: command >> out.txt

Redirect stderr: command 2> err.log

Pipe output to next command: cat file | grep foo | sort | uniq -c
